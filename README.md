TanglabCircularRNAPipeline
==========================
This Pipeline process data from raw fastq data to circular RNA related results.

The following software were used:
Tophat      (v2.0.9)
Cufflinks   (v2.2.1)
HTSeq       (v0.6.0)
CIRCexplorer(v1.0.4)

Running:
  :::bash
  python run\_cirRNA.py [options] sample\_lists
